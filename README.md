Sistema de Préstamos

Sistema de escritorio desarrollado en Java (NetBeans) para la gestión de clientes, empleados, préstamos y pagos dentro de una empresa.

Funcionalidades
Registro de clientes
Registro de empleados
Gestión de préstamos
Control de pagos
Cronograma de pagos
Consulta de información en tablas
Interfaz gráfica desarrollada en Java Swing

Tecnologías utilizadas
Java
NetBeans IDE
MySQL (o el motor que estés usando)
JDBC para conexión a base de datos
Swing para la interfaz gráfica

Arquitectura del proyecto

El sistema está organizado en capas:

Modelo: Clases de entidades (Cliente, Empleado, Préstamo, Pago)
Controlador: Lógica del sistema
Vista: Interfaces gráficas en Swing
Base de datos: Persistencia de información

Base de datos

El sistema maneja tablas como:

clientes
empleados
prestamos
pagos
cronograma

Cómo ejecutar el proyecto
Clonar el repositorio
Abrir el proyecto en NetBeans
Configurar la conexión a la base de datos en el archivo de configuración
Ejecutar la clase principal

Objetivo del proyecto

Este sistema fue desarrollado como proyecto académico para simular la gestión de préstamos empresariales, aplicando conceptos de programación orientada a objetos, bases de datos y desarrollo de interfaces gráficas.
